## Anchor Text Best Practices

1. **Use keyword-rich anchor text** — not "click here" or "read more"
2. **Vary anchor text** — don't always use the exact same phrase
3. **Link deep pages** — game pages, blog posts, achievements (not just top-level nav)
4. **Contextual links** — embed within paragraph copy, not just navigation
5. **3-5 internal links per page** minimum
6. **Reciprocal linking** — if A links to B, B should link back to A somewhere
